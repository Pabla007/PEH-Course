If want to update some software who's update has been come (i.e packages) than we have to use this command and full form of apt is advance package tool.
apt update && apt upgrade


If want to remove something that have to use
apt autoremove


But if we want to install a specific tool/package than we must require apt/pip/pip3 to install it.
apt install python-pip3

If want to install something from github than will change directory to opt (i.e cd opt)
