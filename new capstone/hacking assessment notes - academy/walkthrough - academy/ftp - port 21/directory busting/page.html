- There are 2 tools that will use and discuss the prons & cons of using them.

First : dirb
Command : dirb http://192.168.142.131

Second is : ffuf
ffuf is an acronym for “fuzz 
faster you fool
!”
Command : apt install ffuf
Command : ffuf -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt:FUZZ -u http://192.168.142.131/FUZZ
where -w stands for wordlist
-u stand for url
And will add the FUZZ command with the wordlist and url in order fuzz with it.

As a result we got our results much much faster using ffuf as it is only searching one level deep.
So always remember different Strategy's requires different tools.There are lot of tools that come out for Directory Busting. Latest ones
are being built in RUST and GO. So we have to pick our Poison on directory busting tools and simply run it.
